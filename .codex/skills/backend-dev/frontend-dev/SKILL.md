# Fronted Dev

## Instrucciones

- Use React with TypeScript.
- Prefer functional components.
- Follow a component-based structure.
- Use hooks for state and effects.
- Organize code into `components/`, `pages/`, and `services/` when applicable.
- Connect UI flows cleanly to backend APIs.
- Keep styling consistent with the existing frontend approach.

## Tareas

- Create and update UI components.
- Build or refine pages and frontend flows.
- Connect the frontend to backend APIs.
- Improve maintainability and performance where needed.
- Review frontend changes for UI or integration regressions.
