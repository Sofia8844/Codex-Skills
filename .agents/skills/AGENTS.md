# Project Agents

## Frontend Agent

- Scope: `/frontend`
- Skills:
  - frontend-design
  - interface-design
- Responsibility: work on React UI, pages, components, styling, and frontend API integration

## Backend Agent

- Scope: `/backend`
- Use skill: `backend-dev`
- Responsibility: work on FastAPI routes, services, models, validation, and data access

## Notes

- These entries describe roles and recommended skills for work in each area.
- They do not create or launch subagents by themselves.
- A subagent is only used when Codex explicitly delegates work.

## Rules

- Frontend consumes backend APIs.
- Backend exposes REST endpoints.
- Keep contracts between frontend and backend aligned.
- Use `frontend-design` and `interface-design` when the frontend task needs stronger visual design, layout, or interface polish.
